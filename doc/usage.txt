NAME
       dmq.sh - Manage Distributed Messaging Queue Project

SYNOPSIS
       dmq.sh [OPTION]... [ARG]...

DESCRIPTION
       dmq.sh builds, compiles DMQ project quickly.
       Helps easy interface to project

       -b, --build, build
             Building Code with Source code.
             Example. dmq.sh --build

       -r, --run, run
             Running Code 
             Example. dmq.sh --run no_of_users no_of_servers

       -s, --search, search
             Search certain keyword in source code
             Example. dmq.sh --search user 

       -c, --count, count
            Count no. of lines in source
            Example. dmq.sh --count
